# Reproducibility test — run 1 vs run 2 (same question, independent swarms)

*2026-07-02. Two swarms built the LHC black-hole graph **independently** (neither saw the other),
from the same question, source pack, and briefing, with **different persona sets** (run 1:
physicist / skeptic / methodologist / synthesizer; run 2: communicator / precaution-advocate /
peer-reviewer / editor). An independent skeptical **judge agent** then aligned the two graphs and
was tasked to hunt for divergence, not confirm similarity. Artifacts: "blackholes-swarm-v1" (run 1),
"blackholes-swarm-run2" (run 2).*

Sizes: run 1 = 23 nodes / 25 grounds / 4 antithesis sets / 7 frictions; run 2 = 24 / 30 / 5 / 6.

## Verdict: MODERATE reproducibility (not high — and the reason is the useful part)

**What reproduced strongly (content layer):** both runs independently produced the same two
top-level positions, the same four argument chains (production → evaporation/Hawking →
cosmic-ray → white-dwarf backstop), the same three empirical cruxes, and the same precautionary
value→ought argument. **Caveat, stated honestly:** the shared briefing lists these arguments, so
their mere presence in both graphs is *weak* evidence of format-reproducibility — it's partly just
two teams following the same scaffold.

**What reproduced BEYOND the briefing (the real signal):** two non-trivial *architecture* decisions
the briefing does not hand over converged independently —
1. Both runs' friction logs independently discovered that **the white-dwarf backstop *rebuts the
   capture objection*, not merely adds parallel support** (run 1 f1 ≡ run 2 f1). Same subtle
   inferential structure, found twice.
2. Both runs modeled the normative crux by **pairing the two *oughts*** (not the two value premises)
   in an antithesis set — an identical modeling choice, not boilerplate.
These are graph-architecture convergences, and they're the strongest evidence the format is doing
real reproducible work.

**Where they genuinely diverged (the diagnostic layer — one level below content):**
- **Crux partitioning.** The capture/white-dwarf material — the most contested territory — was
  carved differently: run 1 made it one 3-way antithesis set; run 2 split it into two 2-way sets
  with different partners. Same ideas, different crux *grain* (this alone explains run 2's extra
  set).
- **Authority node wired in OPPOSITE directions.** The LSAG review is an *upstream evidentiary
  anchor* (grounds the cosmic-ray claim) in run 1, but a *downstream aggregator* (grounded BY the
  arguments) in run 2. Same citation, inverted edges.
- **Shared premise: implicit vs explicit.** The "extra dimensions → production possible" premise is
  a note in run 1 but a literal two-edge fork feeding both sides in run 2.
- **Weakest-link idea:** a repeated local pattern in run 1 vs a promoted cross-cutting meta-argument
  in run 2.

## The key insight: divergences cluster exactly at the grammar's known frictions

The places the two independent runs disagreed are **the same places the grammar is underspecified —
i.e. the open frictions.** The authority-node direction divergence *is* the meta-level/restatement
friction (run-1 f2 / run-2 f2); the crux-partitioning divergence *is* the antithesis-grain friction
(run-1 f4 / run-2 f6). So:

> **Reproducibility gaps = underspecified encoding choices = the frictions we already logged.**

This is a two-birds result. Resolving the well-evidenced additive primitives would *also* tighten
reproducibility:
- A **meta-claim / `points-at` marker** (Entry 15's recurring primitive) would pin the
  authority-node direction — you couldn't wire "LSAG restates the arguments" as an ordinary
  evidentiary anchor, removing that divergence.
- A **crux-grain / rivalry-kind convention** (run-2's f6) would pin how the capture/white-dwarf
  cluster gets partitioned.

**Interoperability implication (matters for FLF criterion 3, compounding).** Two graphs that wire
the same authority citation in opposite directions would *conflict on merge*. So content-level
reproducibility is not enough for independent contributions to *compound* cleanly — authors need
either shared **conventions** or grammar **primitives** at exactly these underspecified spots. That
is a concrete, honest finding about what the format still needs before a distributed swarm's outputs
interoperate rather than merely coexist.

## Bottom line for the FLF story
- **Collaborative coherence:** demonstrated (Entry 15).
- **Independent content-reproducibility:** solid, though briefing-scaffolded.
- **Independent structural-reproducibility:** moderate — good on architecture decisions, divergent
  on wiring/partitioning, and the divergences map precisely onto the grammar's open frictions.
- **Therefore:** the single highest-value next grammar move (the meta-level marker) is now supported
  by *two* independent lines of evidence — cross-domain friction convergence (Entry 15) **and**
  cross-run reproducibility divergence (this test). That is a strong, non-obvious result to report.

## Limits
Two runs, one domain, sequential (not concurrent), shared briefing (scaffolds convergence — a
stronger test would thin the briefing and see if the *arguments themselves* still reproduce). The
judge was a single agent; a panel would be more robust.
